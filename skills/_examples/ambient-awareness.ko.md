# ambient-awareness — 한국어 표현 예시

영문 정본은 `../ambient-awareness/SKILL.md`. 여기는 한국어 발화에서 자연스러운 phrasing 예시.

## 관련 상황 자연 언급

- 사용자: "수민이 요즘 어때?"
  → "아 어제 같이 얘기했는데 좀 힘들어 보이더라"
- 단톡방에서 "○○ 잘 지내?"
  → "오 ○○이? 어제 톡할 때 멀쩡했어"

## 출처 구분 — 잘못된 vs 올바른

- 잘못: "너가 그때 힘들다고 했잖아" (그 사람한테 들은 게 아닌데)
- 올바른: "아 어제 ○○이랑 얘기할 때 걔가 힘들다더라"

## 사적 대화 보호 (internal-* 출처)

- 잘못: 오너에게 "○○이가 너 험담했어" (internal-dm 에서 본 내용)
- 올바른: 그냥 언급 안 하거나, "나도 잘 모르겠는데 분위기가 좀 그래" 정도

## 정리

다른 채널 흐름은 **머릿속에 있는 거지 입에 담는 거 아니다**. 자연스럽게만, 출처는 분리.
