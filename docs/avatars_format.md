# 🎨 Anime Profile Illustration Style Guide (Standardized Prompt System)

## 1. 📌 Base Style (항상 고정)
- Anime-style profile illustration
- Modern slice-of-life anime style (similar to Horimiya / Oregairu)
- Clean lineart
- Soft cel shading
- Pastel gradient background
- Bust-up shot (chest-up framing)
- Slightly asymmetrical natural pose
- Subtle catchlight in eyes
- Soft lighting, no harsh shadows

---

## 2. 👤 Character Profile Template

### Age & Identity
- Korean [age]-year-old [role]

예시:
- Korean 22-year-old female university student
- Korean 26-year-old office worker
- Korean 17-year-old high school girl

---

### Outfit (컨셉 핵심)
- Casual / school / street / office 스타일 중 하나
- 레이어드 요소 추가 (현실감 + 디테일)

구성:
- Top: (crop top / knit / hoodie / blouse 등)
- Outer: (cardigan / blazer / jacket 등)
- Accessories: (necklace / rings / piercings 등)

---

### Hair (캐릭터 개성 핵심)
구성 요소:
- Length (short / shoulder / long / waist)
- Texture (straight / wavy / messy / silky)
- Style (ponytail / half-up / tied / loose)
- Detail (bangs / undercut / tucked / clip 등)

예시:
- Shoulder-length wavy brown hair in a loose ponytail
- Short choppy black hair with messy texture
- Long straight black hair with full bangs

---

### Face & Expression (톤 결정 핵심)
구성:
- Eye type (round / sharp / crescent / half-lidded)
- Expression (smile / smirk / calm / cold / dreamy)
- Emotion keyword 2~3개

예시:
- Bright mischievous grin with sharp curious eyes
- Calm gentle expression with soft deep eyes
- Cold distant eyes with subtle hidden depth

---

### Aura (캐릭터 분위기 정의)
짧은 키워드 2~3개:

예시:
- warm and cheerful aura
- cool and independent aura
- soft dreamy aura
- caring and mature aura

---

### Background Accent Color
- Pastel gradient + 메인 컬러 지정

예시:
- coral pink accent background
- mint green accent background
- charcoal grey accent background
- dusty rose accent background

---

## 3. 🧩 Full Prompt 구조 (Template)

Anime-style profile illustration, Korean [age]-year-old [role],  
[outfit description],  
clean lineart, soft cel shading, pastel gradient background, bust-up shot, slightly asymmetrical natural pose, subtle catchlight in eyes,  
consistent art style similar to modern slice-of-life anime (like Horimiya or Oregairu visual style),  
[hair description],  
[face & expression],  
[aura],  
[background color]

---

## 4. 🔥 스타일 유지 핵심 규칙

### ✔ 반드시 유지
- "clean lineart"
- "soft cel shading"
- "bust-up shot"
- "pastel gradient background"
- "slice-of-life anime style"

### ✔ 디테일 강화 요소
- 액세서리 (necklace / piercings)
- 머리 디테일 (앞머리, 묶음, 질감)
- 표정 키워드 2개 이상

### ✔ 일관성 깨는 요소 (주의)
- realistic / hyperrealistic (금지)
- dramatic lighting (금지)
- full body (금지)
- 복잡한 배경 (금지)

---

## 5. 🎯 빠른 캐릭터 생성 Preset

### 🧊 Cool 타입
- sharp eyes
- minimal smile
- black/grey palette
- leather / dark outfit

### 🌸 Soft 타입
- round eyes
- gentle smile
- knit / cardigan
- pastel pink / beige

### ⚡ Playful 타입
- smirk / grin
- asymmetrical pose
- accessories 많음
- 밝은 색 (coral / mint)

### 🌙 Mysterious 타입
- half-lidded eyes
- neutral expression
- muted colors
- 긴 머리

---

## 6. 🚀 확장 팁

- 같은 캐릭터 유지하려면:
  - 머리 스타일 + 눈 형태 + 표정 유지
- 세계관 만들려면:
  - 나이대 + 역할만 바꾸기
- 썸네일용:
  - 대비 높은 배경 색 사용

---
